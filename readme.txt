With the Proweblook Phone Validator plugin you can easily verify if a phone number is really valid and callable (https://proweblook.com).

== Description ==

This plugin uses the Proweblook Phone-Validator API (https://proweblook.com) to check if a phone number is really valid and callable.

= Supported Plugins / Compatible Forms =

The plugin works seamlessly with:
* Contact Form 7
* Gravity Forms
* WPForms
* Ninja Forms
* WooCommerce

The plugin can also be easily integrated into any 3rd party form.

= Key Features =

How does it work?

The Phone-Validator real-time validation API provides detailed information for each phone number:
* Status: VALID/CONFIRMED or INVALID
* Line type: FIXED_LINE, MOBILE, VOIP etc.
* Geolocation: Region and city
* Phone number correction and re-formatting according to national and international standards

During the entire validation process, the phone number is not contacted in any way, nor will any phones connected to a line ring.

== Installation ==

1. Unzip and upload the plugin to the '/wp-content/plugins/' directory.
2. Activate the plugin through the 'Plugins' menu in WordPress.
3. Configure the plugin in the Settings section of the Administrator Dashboard.
4. Register for an API key and get 10 free credits (no credit card required):
https://proweblook.com/auth/signup
5. Copy and paste your API key to the Settings section of the Administrator Dashboard.
